  parameterName,

Get a parameter handle to a parameter, located in a specific category.

If the category argument is None the function will look into all categories and return the first parameter matching "parameterName".

Parameters
pluginHandleHandle to a plugin
parameterNameName of the parameter
categoriesNameName of a category containing the parameter.
Returns
Returns a handle to a parameter. Returns None if not found
