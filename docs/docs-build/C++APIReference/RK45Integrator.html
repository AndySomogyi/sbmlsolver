double integrate(double t, double h)¶

implement Integrator interface

Integrates the model from t to t + h. Attempts to find the state vector at t + h, and returns time t + h if successful. If the integrator does not find a convergent solution, the state vector is not updated and t is returned. Also calculates a new timestep and saves it to a member variable for future use.
