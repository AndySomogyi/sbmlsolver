\doxysubsection{Detailed Description}
A super basic 4\textquotesingle{}th order fixed step integrator. 

The Runge\+Kutta\+Integrator will be the more sophisticated, general purpose Runge-\/\+Kutta integrator which will support different orders and adaptive time stepping.

This object is mainly here as an example of creating a new \mbox{\hyperlink{classrr_1_1Integrator}{Integrator}}.

Creates a new \mbox{\hyperlink{classrr_1_1RK4Integrator}{R\+K4\+Integrator}}. 

The \mbox{\hyperlink{classrr_1_1IntegratorFactory}{Integrator\+Factory}} is the O\+N\+LY object that creates integrators.

Integrators are created when the \mbox{\hyperlink{classrr_1_1IntegratorFactory_ab7298b70afb06e1200181cf3981f739c}{Integrator\+Factory\+::\+New}} method is called, typically by the top level \mbox{\hyperlink{classrr_1_1RoadRunner}{Road\+Runner}} object.

The integrator will hold onto the \mbox{\hyperlink{classrr_1_1ExecutableModel}{Executable\+Model}} pointer, m, and when the integrate method is called, will advance the model object forward in time.
