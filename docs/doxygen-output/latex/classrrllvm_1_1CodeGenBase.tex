\subsubsection*{template$<$typename Function\+Ptr\+Type$>$\newline
class rrllvm\+::\+Code\+Gen\+Base$<$ Function\+Ptr\+Type $>$}
a convenience class to pull the vars out of a context, and store them as ivars. 

It can get tedious alwasy typing mgc.\+get\+This and mgc.\+get\+That. Furthermore, its faster to access them as ivars as it does not incur a func call each time.
