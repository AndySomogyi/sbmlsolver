\doxysubsection{Detailed Description}
Hash table designed for storing \mbox{\hyperlink{classrr_1_1RoadRunner}{Road\+Runner}} models. 

Expensive operations like building models are done in parallel.

Loading \mbox{\hyperlink{classrr_1_1RoadRunner}{Road\+Runner}} models requires expensive builds, which can be prohibitive when loading many models. \mbox{\hyperlink{classrr_1_1RoadRunnerMap}{Road\+Runner\+Map}} is a container for \mbox{\hyperlink{classrr_1_1RoadRunner}{Road\+Runner}} objects which abstracts the process of loading or simulating in parallel.
