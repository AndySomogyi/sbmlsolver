\doxysubsection{Detailed Description}
The routine computes the singular value decomposition (\mbox{\hyperlink{classrr_1_1SVD}{SVD}}) of a real m-\/by-\/n matrix. 

Often when computing the steady state of a system the jacobian matrix is singular. \mbox{\hyperlink{classrr_1_1SVD}{SVD}} is a way of computing the rank of a matrix, which in turn can be used to determine whether a rank is singular\+:
\begin{DoxyItemize}
\item if rank \texorpdfstring{$<$}{<} min(\+N, M) of an NxM matrix then it is singular. 
\end{DoxyItemize}
