\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classrrllvm_1_1EventQueue}{Event\+Queue}} class is when events are queued to execute, but have not yet executed. Some events may be \textquotesingle{}de-\/listed\textquotesingle{} from the \mbox{\hyperlink{classrrllvm_1_1Event}{Event}} Queue if thier trigger changes back to \textquotesingle{}false\textquotesingle{}, if they are set persistent=False.
