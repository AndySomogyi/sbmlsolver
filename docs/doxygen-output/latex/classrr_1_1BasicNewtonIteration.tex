\doxysubsection{Detailed Description}
Solve for steady state using Kinsol\textquotesingle{}s implementation of Newton Iteration. 

The kinsol docs refer to this algorithm as the \char`\"{}basic\char`\"{} newton iteration to differentiate it from the \char`\"{}linesearch\char`\"{} strategy which is a variant of the \char`\"{}basic\char`\"{} newton iteration. \begin{DoxySeeAlso}{See also}
