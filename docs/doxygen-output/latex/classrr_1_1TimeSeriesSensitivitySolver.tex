simulate a timeseries with sensitivities from start to step with num data points. 

\mbox{\hyperlink{classrr_1_1Matrix3D}{Matrix3D}} indexed by time. Each element of the 3D matrix is a Matrix$<$double$>$ with rows and columns parameters and model variables respectively. The parameter k determines the kth order derivative of the sensitivity information that will be returned 
\begin{DoxyParams}{Parameters}
{\em start} & starting time for time series simulation \\
\hline
{\em stop} & last time point for time series simulation \\
\hline
{\em num} & number of data points to simulate. Determines Z of \mbox{\hyperlink{classrr_1_1Matrix3D}{Matrix3D}}. \\
\hline
{\em params} & vector of parameters that you want sensitivity for. When empty (default), compute sensitivities for all parameters vs all variables. \\
\hline
{\em species} & vector of species to include in the results. Note that unlike for params, reducing the number of species does not solve faster because the matrix slicing only occurs at the end. When empty (default) all species are returned. \\
\hline
{\em k} & (default 0) return the kth other derivative of the sensitivity data. \\
\hline
\end{DoxyParams}
