\doxysubsection{Detailed Description}
Constructs new integrators. 

\begin{DoxyAuthor}{Author}

\end{DoxyAuthor}
Implements the factory and singleton patterns. Constructs a new integrator given the name (e.\+g. cvode, gillespie) and returns a base pointer to \mbox{\hyperlink{classrr_1_1Integrator}{rr\+::\+Integrator}}.
