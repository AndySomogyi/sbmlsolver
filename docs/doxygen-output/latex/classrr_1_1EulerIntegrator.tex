\doxysubsection{Detailed Description}
An example class intended to show how to create an \mbox{\hyperlink{classrr_1_1Integrator}{Integrator}}. 

This class implements the most basic possible integration algorithm\+: the forward Euler method.

This integrator should not really be used in practice as the forward Euler algorithm is highly unstable and will seldomly yield numerically correct values.

An example of calling simulate in Python is


\begin{DoxyCode}{0}
\DoxyCodeLine{rr.setIntegrator(\textcolor{stringliteral}{'euler'})}
\DoxyCodeLine{rr.simulate();}

The \mbox{\hyperlink{classrr_1_1IntegratorFactory}{Integrator\+Factory}} is the O\+N\+LY object that creates integrators.

Integrators are created when the \mbox{\hyperlink{classrr_1_1RegistrationFactory_af6a9a68a5895f4da2bf3f57872b09bd4}{Integrator\+Factory\+::\+New}} method is called, typically by the top level \mbox{\hyperlink{classrr_1_1RoadRunner}{Road\+Runner}} object.

The integrator will hold onto the \mbox{\hyperlink{classrr_1_1ExecutableModel}{Executable\+Model}} pointer, m, and when the integrate method is called, will advance the model object forward in time.
