\doxysubsection{Detailed Description}
Base class for all code generation systems; allows compiling and evaluating the model. 

The \mbox{\hyperlink{classrr_1_1ExecutableModel}{Executable\+Model}} interface provides a way to access an sbml model that was compiled, J\+IT\textquotesingle{}d or interpreted as executable (runnable) module.

An \mbox{\hyperlink{classrr_1_1ExecutableModel}{Executable\+Model}} holds a Model\+Data structure, all the simulation values are stored in the Model\+Data struct, i.\+e. the dynamic state of the model is fully contained in the Model\+Data structure.

An \mbox{\hyperlink{classrr_1_1ExecutableModel}{Executable\+Model}} shoud also contain all of the initial condisions, rules, functions and whatever other semantic information that was specified in the sbml model.
