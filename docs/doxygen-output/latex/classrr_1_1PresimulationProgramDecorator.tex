\doxysubsection{Detailed Description}
wrapper class to change the solve() method of \mbox{\hyperlink{classrr_1_1SteadyStateSolver}{Steady\+State\+Solver}} types. 

Pass a solver to the \mbox{\hyperlink{classrr_1_1PresimulationProgramDecorator}{Presimulation\+Program\+Decorator}}. The resulting object can be used as a \mbox{\hyperlink{classrr_1_1SteadyStateSolver}{Steady\+State\+Solver}}; the solve() method of which will apply presimulation to a steady\+State calculation.
