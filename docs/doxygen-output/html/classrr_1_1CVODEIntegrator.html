Detailed Description

A RoadRunner integrator based on CVODE; serves as RoadRunner's main integrator for ODEs.

Author
WBC, ETS, MTK Provides a wrapper around CVODE from the SUNDIALS suite. CVODE has two main solvers: an Adams-Moulton solver for non-stiff problems, and a backward differentiation formula (BDF) solver for stiff problems. See: https://computation.llnl.gov/casc/sundials/documentation/toms_sundials.pdf
