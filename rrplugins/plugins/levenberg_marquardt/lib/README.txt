This folder contain files needed to compile a static version of the lmfit C library.
The actual version of the library being used is found in the file CHANGELOG.

The CMakeLists.txt file in this folder is not part of the lmfit library. It was created by Totte Karlsson (tottek@gmail.com)
in order to be used with the Tellurium plugin framework.

The versions folder contain original downloaded sources from lmfit's webpage at http://apps.jcns.fz-juelich.de/doku/sc/lmfit
