Tellurium PluginAPI Python Wrapper

The telplugins_c_api.py file is a python wrapper around the telplugins_c_api library.

On top of the raw API defined by telplugins_c_api.py, is another more user friendly API defined, in the file telplugins.py.

Necessary files in this folder should be installed to your systems python installations site-packages folder.
