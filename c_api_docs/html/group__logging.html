Get the logging status level as a pointer to a string.

The logging level can be one of the following strings

"ANY", "DEBUG5", "DEBUG4", "DEBUG3", "DEBUG2", "DEBUG1", "DEBUG", "INFO", "WARNING", "ERROR"
